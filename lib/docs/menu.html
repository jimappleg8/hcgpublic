Menu Module

left_menu

<?php get("menu.left_menu", $item_id, [$root_level], [$hilitepath], [$tpl]); ?>

The "root_level" number determines how many parents are displayed as part of the menu. The default is "2" with shows just the submenu for the section you're in. A level of "1" will include all sections in the menu. Regardless of the "root_level" the direct line of pages from the home page is always included. If you don't want the upper levels to be displayed, you will need to set display: none in the style sheet for those items.

Description

This version of the left_menu tag uses a flat file store in <doc_root>/inc/<site_id>_menu.txt for the source info about the menu.
