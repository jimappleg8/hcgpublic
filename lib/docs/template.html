Template Module

include_tpl(

<?php get("module.tag_name", $tpl_file, [$data_array], [$data_name]); ?>

Description

Returns the specified template, optionally replacing any variables in the template with those supplied in $data_array. $data_array is an associative array. The tag will assign the data to a variable based on the template name unless a $data_name is specified. For example, in the header.tpl file, variables can be accessed by {$header.variable_name}.

include_tpl_array

<?php get("module.tag_name", $tpl_file, $data_array); ?>

Description

Similar to the include_tpl tag except it allows for multiple assign statements. The data array is a two-dimensional array in the form array('label1' => array('key1' => "value1"), 'label2' => array('key2' => "value2"). In the template you would be able to access these as {label1.key1} and {label2.key2}
