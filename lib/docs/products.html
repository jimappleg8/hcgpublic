Products Module

category_id_title

<?php get("products.category_id_title", $cat_id); ?>

Description

Returns the category name in title case, based on the Category ID.

category_list

<?php get("products.category_id_title", $cat_id, [$site_id], [$tpl]); ?>

Description

Displays all products in a given category.

get_brand_name

<?php get("products.get_brand_name", [$site_id]); ?>

Description

Returns the brand name given a site ID. If no site ID is given, it uses the current site.

multi_category_list

<?php get("products.multi_category_list", $cat_id, [$cat_list], [$site_id], [$tpl]); ?>

Description

Displays all products in a given category, sorted by a second category. For example, it can display a list of kosher teas sorted by tea type (i.e. green tea, herb tea, red tea, etc.).

Template Variables

The tag looks for multi_category_list.tpl as the default template.

multi_category_list_by_field

<?php get("products.multi_category_list_by_field", $select, $category, [$cat_list], [$site_id], [$tpl]); ?>

Description

Similar to the multi-category_list except the primary "category" is based on an SQL "where" statement. You can essentially create a new category by searching for a set of records that meet a certain criteria. For example, if we want to list all new products, we can use a $select of "FlagAsNew = 1".

nutrition_facts

<?php get("products.nutrition_facts", $prod_id, [$display_hd]); ?>

Description

Returns a formatted HTML version of a product's Nutrition Facts panel.

Template Variables

The tag uses one of several templates depending on what type of Nutrition Facts box the product has. For example, baby food Nutrition Facts boxes are different than the standard one. As of now, the only template that is complete is the standard US nutrition facts, but more will be completed shortly, including a Canadian version.

product_detail

<?php get("products.product_detail", $prod_id, [$cat_id], [$site_id], [$tpl]); ?>

Description

No description.

Template Variables

The tag looks for product_detail.tpl as the default template.

product_meta

<?php get("products.product_meta", $prod_id); ?>

Description

Returns just the product's meta data in a format that allows the information to be passed to a template as a variable.

random_product

<?php get("products.random_product", [$bool_field], [$cat_id], [$site_id]); ?>

Description

Returns product data for a random product for the brand, optionally limited to products where a boolean field is set to true (such as "FlagAsNew") or products in a particular category. Can be used to display a "featured product" on any page of the site.
