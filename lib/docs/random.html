Random Module

random_quote

<?php [echo] get("random.random_quote", ["site_id"]); ?>

Description

Returns a random quote or other line of text from a file: inc/quotes.txt". The quote is returned rather than printed directly (echo'd) in case it is to be used in a template and needs to be passed as a variable.

This tag can return any string from the /inc/quotes.txt file, so it could be used for other purposes such as a random hint of the day.

The site ID can be overridden so the quote is pulled from another site's quote file. This makes it possible for multiple sites to share a single quote file.

Template Variables

There is no template for this tag.

random_image

<?php [echo] get("random.random_image", ["site_id"]); ?>

Description

Returns a random image filename and the "alt" text for the image from a file: <doc_root>/inc/<site_id>_images.txt". The file must contain filenames, one per line, in the form /path/to/file/filename.jpg|Image alt text.

The filename is returned rather than printed directly (echo'd) in case it is to be used in a template and needs to be passed as a variable.

The site ID can be overridden so the filename is pulled from another site's quote file. This makes it possible for multiple sites to share a single image file.

Template Variables

There is no template for this tag.
