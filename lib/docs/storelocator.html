StoreLocator Module

all_locator_results

<?php get("storelocator.all_locator_results", [$site_id], [$tpl]); ?>

Description

Displays stores near a given zip code using our both our local store database and the IRI database.

iri_locator_results

<?php get("storelocator.iri_locator_results", [$site_id], [$tpl]); ?>

Description

Displays stores near a given zip code using the IRI store database. This database is accessed remotely via XML.

local_locator_results

<?php get("storelocator.local_locator_results", [$site_id], [$tpl]); ?>

Description

Displays stores near a given zip code using our local store database.

store_locator_form

<?php get("storelocator.store_locator_form", [$loc_code], [$site_id], [$tpl]); ?>

Description

Displays the store locator form so people can enter their zip code and get a list off stores.
