MailForm Module

contactus_form

<?php get("mailform.contactus_form", $mailtpl1, $mailtpl2); ?>

Description

Displays and processes the standard "Contact Us" form.

The tag allows for up to two emails to go out, usually one to the user and another to an internal recipient. To disable one or either of these e-mails, simply send an empty string for $mailtpl1 and/or $mailtpl2. If neither e-mail is sent, the data is just saved to the database.

disclaimer_alert

<?php get("mailform.disclaimer_alert"); ?>

Description

Inserts the JavaScript for disclaimer if set to display in the lib/config.inc.php file.

This is a disclaimer we needed to alert people that our consumer relations group was behind on e-mails and that it might take a while to respond. We wanted to be able to turn it on and off easily, thus the tags.
