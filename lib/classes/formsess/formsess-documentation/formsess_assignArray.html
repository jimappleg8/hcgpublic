assignArray()

void assignArray($array, $force);
array $array;
bool $force;

Assigns a list of values to a list of fields.

Very useful when you need to assign a lot of fields, by instance when showing data taken from a database in an edit form, you can easily fetch the data in an associative array and use assignArray to fill in a list of fields.

The $array is an associative array built like 'fieldname' => 'fieldvalue'. The fieldname and fieldvalues are the same than what you would have given to the assign function.

See also assign
