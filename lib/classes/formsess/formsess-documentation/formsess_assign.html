assign()

void assign( $field,  
  $value,  
  $force);  
string   $field;
string   $value;
bool   $force;

Assigns a value to a field.

Assigns $value to the field $field. If a value is already assigned to a field, you can force overwritting using $force and setting it to true.

The assigned value is usually a string, integer or boolean. However, a few fields can accept an array as $value: select multiple, checkbox and radio. For these ones, any item in the array with a matching value will be selected / checked.

See also forget_field
