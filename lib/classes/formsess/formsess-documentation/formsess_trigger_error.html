trigger_error()

void trigger_error( $field,  
  $error_message);  
string   $field;
string   $error_message;

Add an error for a field.

Very useful when you want to run some custom server side validation and benefit of formsess error messages mechanisms. Just trigger an error on any field you want with the error message of your choice, and it will be displayed as if it had been triggered by a built-in validator.

If a value was previously saved for that field, it is erased.

The $error_code parameter is obsolete.

See also reset, hasErrors
