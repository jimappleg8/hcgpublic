addCheck()

void addCheck( $check,  
  $fields,  
  $params,  
  $message);  
string   $check;
string   $fields;
array   $params;
string   $message;

Adds a validation which will be performed when calling performCheck(). addCheck() is usually not used directly; it is supposed to be called when using <fs:check />

$check is the validator name, $fields a field name or a list of field names, $params an associative array of parameters used by the validator, and $message the error message displayed when an error occurs.

See also performCheck
