<fs:input />

fs:input is similar to (X)HTML's input tag. Depending on the type parameter, a standard input control is displayed.

Parameters:

  • name: name of the form field
  • type: input type (text, hidden, password, checkbox, radio)
  • value: mandatory for radio and checkbox

Any other parameter that applies to the matching (X)HTML control is valid for <fs:input />

Sample: 


TEXT
<fs:input type="text" name="foo" maxlength="20" />

HIDDEN
<fs:input type="hidden" name="hiddenfoo" />

PASSWORD
<fs:input type="password" name="passfoo" class="someclass" />

RADIO
<fs:input type="radio" name="radiofoo" value="1" />
<fs:input type="radio" name="radiofoo" value="2" />
<fs:input type="radio" name="radiofoo" value="3" />

CHECKBOX
<fs:input type="checkbox" name="radiochk" value="1" />
<fs:input type="checkbox" name="radiochk" value="2" />
<fs:input type="checkbox" name="radiochk" value="3" />
