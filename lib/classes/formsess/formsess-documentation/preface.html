Introduction

Formsess is a forms toolkit designed for Smarty, the compiling template engine.

It handles most common step of forms manipulation:

  • generation
  • data association
  • validation (server and client side)
  • persistance of data through session

Even if several solutions aiming to the same goal already exist, formsess was written in order to use as much as possible smarty features (prefiltering and back/frontend separation). It provides an elegant syntax allowing you to keep full control on forms & validation output.
