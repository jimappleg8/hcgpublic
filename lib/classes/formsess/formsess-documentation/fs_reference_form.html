<fs:form />

The fs:form tag is the formsess equivalent of the (X)HTML form tag.

You have to surround any form with that tag.

Parameters:

  • action: action of the HTML form (e.g. target script) (mandatory)
  • method: HTTP method used to send the data (POST / GET) (mandatory)
  • name: name of the HTML Form. Very important param, since it is also the name you will use to manipulate your form from your PHP Script

All the standard HTML parameters that apply to the FORM tag are also valid.

Sample: 


<fs:form name="foo" action="bar.php" method="POST">
	<!-- form code -->
</fs:form>
