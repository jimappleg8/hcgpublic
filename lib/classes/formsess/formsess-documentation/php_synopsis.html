Chapter 4. Synopsis

Most formsess operations in the PHP Code depend on the formsess class, defined in formsess.class.php.

We will starting from now consider two types of PHP Scripts:

  • Form scripts: They are used to display a form. They don't perform any server side validation.
  • Action scripts: This is the place where the data sent using the forms is handled. Usually the action parameter of the form

Of course you can work without splitting the roles that way, but in my opinion it makes the explanations easier and more easy to understand.
