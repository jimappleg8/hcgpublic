Chapter 2. Requirements

Smarty.  Of course, since Formsess is designed for Smarty, the compiling template engine, it requires smarty to be used. The exact version of smarty formsess requires is unfortunately not known, since I didn't test all the features with older versions of Smarty. It works with 2.5.0 and more. If you get results with older versions, please send me an email so that i can update the documentation.

PEAR::Validate.  Some of the validators require PEAR::Validate to run. They are not listed here yet, but it will be done shortly.
