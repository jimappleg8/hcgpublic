<fs:option />

fs:option is similar to the standard (X)HTML option tag. It adds an option to a <fs:select /> control.

Parameters:

  • value: value of the current option (mandatory)

Any other parameter that applies to the standard (X)HTML option is valid for fs:option.

Sample: 


<fs:select name="foo">
  <fs:option value="1">one</fs:option>
  <fs:option value="2">two</fs:option>
  <fs:option value="3">three</fs:option>
</fs:select>

See also <fs:options />, <fs:select />
