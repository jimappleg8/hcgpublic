<fs:ifErrormessage />

Allows you to control HTML formatting around error messages. By instance when you want to add a row to a table to display your error message, you don't want to display an empty row when there is no error message. This tag allows you to condionnally display HTML Code when an error message exists for a specific field.

Parameters:

  • field: name of the field you want to test the existence of an error message for

Sample: 


<fs:ifErrormessage field="foo"><fs:errormessage field="foo" /></fs:ifErrormessage>

See also <fs:errormessage />
