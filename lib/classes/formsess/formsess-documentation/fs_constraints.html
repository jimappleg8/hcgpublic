Formsess introduces a few constraints you have to follow when developping formsess driven web forms.

Tags have to be closed

As specified in XHTML, all the tags have to be closed. The following tag declaration is not valid with formsess: 


<fs:input type="text" name="foo">

The tag has to be closed or it will not be recognized by the template preprocessor: 


<fs:input type="text" name="foo" />
