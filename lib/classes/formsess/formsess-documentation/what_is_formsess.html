Chapter 1. What is formsess

Formsess is a forms toolkit designed for Smarty, the compiling template engine.

It helps you integrating forms in your web applications without having to write the same code again and again.

The following steps are featured:

  • form generation
  • client side validation (javascript)
  • server side validation
  • custom error messages without any constraint on the error message location
  • data persistance (entered values, error messages, ...)
