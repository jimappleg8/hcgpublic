Chapter 5. Using formsess

From the PHP code, most formsess operations are executed using an instance of the formsess class.

When creating an instance of formsess, two parameters are possible: the form name, and the method. The form name is the string you are using in your template as the name attribute of your <fs:form> tag. If your form is using the POST method, you do not need any additionnal parameter. If it is a GET form, you have to specify the method. 


$fs = new formsess('formname', 'GET');

When your object is created, any method you use on the object refers to the form you have given as a parameter.

In your form script, if you do not need to perform any specific operation on the form from the script, you do not need to create a formsess object.
