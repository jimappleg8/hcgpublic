<fs:html_select_date />

This is a custom tag that was inspired by smarty's html_select_date function. It allows to quickly display 3 form items to select a date in a form. That way, it's much easier to use and validate a date, since any formsess operation (formsess::get_value, or formsess::save) will apply to the date itself and not each field independentely.

Any parameter that can be used with the smarty function html_select_date is valid for this tag. The only difference is that the "prefix" parameter is replaced by "name" in order to match the rest of the formsess architecture.

Sample: 


<fs:html_select_date name="foo" />

See also <fs:html_select_time />
