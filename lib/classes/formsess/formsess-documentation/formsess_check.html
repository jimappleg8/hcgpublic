check()

void check( $check,  
  $fields,  
  $params,  
  $message);  
string   $check;
string   $fields;
array   $params;
string   $message;

check is a very important function in formsess. It is one of the basic methods used to perform server side validation.

Just call it using the validator name as $check, the field to validate as $fields (or an array of field names if the validator needs several fields). If the validator needs additionnal parameters, you can specify them using the $params array, and you can also specify a custom error message using $messsage

See also hasErrors
