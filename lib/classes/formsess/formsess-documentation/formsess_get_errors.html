get_errors()

array get_errors($field);
string $field;

Returns the errors stored for the current form.

If a field name is specificed as $field, the errors are only returned for the given field.

Usually you don't have to use that function since <fs:errormessage /> was introduced to make the integration of error messages as easy as possible.
