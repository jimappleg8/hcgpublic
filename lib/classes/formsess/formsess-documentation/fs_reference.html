This is the formsess templates tag reference.

<fs:errormessage />

Errormessage is the tag used to display an error message for a field.

Parameters:

  • field name of the field to display the error for (mandatory)

Sample: 


<fs:errormessage field="foo" />

See also <fs:ifErrormessage />
