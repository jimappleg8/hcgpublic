No try and error...
patUser supplies a lot of functions for error handling. Most of patUser's method return "false" if any error occured. If you need more information, on why your request to patUser could not be processd, we included several functions. You may either request the last error code, message or both.
But patUser also keeps track of errors that occured before the last function call. We included getAllErrorCodes(), getAllErrorMessages() and getAllErrors() to retrieve the list of all errors.
To modify the errorlist, you may use removeLastError() or clearErrors().
patUser Error Codes
To translate an error code into a readable error message you may either use patUser's translateErrorCode() or write your own translator, we've included documentation of all possible codes at the end of this page.
API documentation
These functions are those available for the above features:

remove the last error from the list



void clearErrors

clear all errors

Returns:

get the last error code AND message

Returns:

array$error last error code AND message


array getAllErrorCodes

get all error codes

Returns:

array$errors array containing all error codes


array getAllErrorMessages

get all error messages

Returns:

array$errors array containing all error messages


array getAllErrors

Returns:

array$errors array containing all error codes AND messages
