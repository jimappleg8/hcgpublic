Miscellaneous functions
There are two more features that do not fit in any before mentioned topics.
Getting information about tables
If you need to get information about a MySql table, just try our getTableInfo(). It will return a lot of information about all fields and possible values in a table.

array getTableInfo

get information about table fields

like DESCRIBE [table]
Parameters:
string$tablenameinternal table name

Returns:

array$desc table description
Re-using dbc objects
With the sendQuery() function you are able to use patUsers internal database connectivities. The first parameter is the internal name of the dbc and the second is a MySql query. The function will return a patDbc result object.

object patDbcResult &sendQuery

send query to any database

can be used the use the internal dbcs in your own apps without knowing how the user object was configured
Parameters:
string$dbcname of the database

string$queryquery

Returns:

object patDbcResult$resultpatDbc result of the query
