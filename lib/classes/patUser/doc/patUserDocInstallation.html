Installation quick guide
Installing patUser is quite easy. Just download your prefered archive type and extract all files.
You will need a mySql database, if patUser should work properly. If you do not already have a database with tables for userdata you will find an sql file in the sql folder, which helps you setting up a database. Use phpMyAdmin to execute the sql statements in the file.
Now just copy all files from the include and templates folder in your project directory and start working .
