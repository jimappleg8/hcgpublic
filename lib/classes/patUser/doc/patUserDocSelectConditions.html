Building select conditions
Several of patUser's functions, like searchUsers() or getUsers() require a list of conditions for users. This section will explain what's possible in these conditions.
General structure
When defining conditions these conditions are always organized in one array. each entry of the array contains one condition. Each condition has to be an associative array, which contains the following keys:


field
This value defines the name of the field in the mySql table. You may use any field that exists in the specified table.


value
Here you define the value for the field. How the fieldvalue will be compared to the value given by this parameter will be defined by the third parameter.


match
This parameter defines the comparison method for condition. Possible values are:


greater (>) : the field value has to be greater than the specified value


greaterorequal (>=) : the field value has to be greater than or equal to the specified value


lower (<) : the field value has to be lower than the specified value


lowerorequal (<=) : the field value has to be lower than or equal to the specified value


like : comparison will be case-insensitive


contains : the field value has to contain the defined value


startswith : the field value has to start with the defined value


endswith : the field value has to end with the defined value


between : when using this method, the value has to be an array that consists of two values. The field value has to be between these values


exact (default): case sensistive comparison is used
Examples
To get all users, whose surname starts with an 'p', use this condition: 
$condition1 = array( "field" => "surname",
                     "value" => "p",
                     "match" => "startswith" );
To get all users, who were at least one hour on your site use this (of course only if stats are enabled): 
$condition2 = array( "field" => "time_online",
                     "value" => 3600,
                     "match" => "greaterorequal" );
Of course you can combine both conditions: 
$users = $user->getUsers( array( "uid", "username" ),
                          array( $condition1, $condition2 ),
                          array( "conditions" => "all") );
This will return the ids and names of all users that were at least one hour online and whose name starts with the letter "p";
